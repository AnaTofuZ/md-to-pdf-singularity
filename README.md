# md-to-pdf singularity def

https://github.com/simonhaenisch/md-to-pdf

# SYNOPSIS

## build

```shell
$singularity build --fakeroot  md-to-pdf.sif md-to-pdf.def
```

## run

```shell
$./md-to-pdf.sif file.md --as-html
```
